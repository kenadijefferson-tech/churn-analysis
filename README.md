# IT Asset Churn Analysis Project

## Overview
This project analyzes customer churn behavior using simulated telecom-style data. The goal is to identify patterns and trends that influence customer retention.

## Objectives
- Identify churn distribution
- Analyze churn by contract type
- Examine the relationship between tenure and churn
- Explore how monthly charges relate to customer behavior

## Tools Used
- Python (Pandas, Matplotlib)
- VS Code
- CSV dataset

## Key Insights
- Customers on month-to-month contracts show higher churn rates
- Shorter tenure is associated with increased likelihood of churn
- Contract type appears to be a strong predictor of retention
- Monthly charges may influence churn behavior

## Visualizations
- Churn distribution
- Churn by contract type
- Tenure vs churn
- Monthly charges distribution

## Project Structure
data/churn.csv → Dataset  
analysis/analysis.py → Analysis script  
visuals/ → Generated charts  

## How to Run
1. Install dependencies:
   pip install pandas matplotlib

2. Run the script:
   python analysis/analysis.py

3. View generated charts in the visuals folder

## Purpose
This project demonstrates data analysis, visualization, and basic business insights relevant to customer retention strategies.
